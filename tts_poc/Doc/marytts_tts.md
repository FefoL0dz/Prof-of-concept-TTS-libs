# MaryTTS Provider Implementation Guide

## Prerequisites
1. A running **MaryTTS** server exposing a REST endpoint.
2. The endpoint URL (e.g., `https://my-marytts.com/api/tts`).

## Steps
1. **Set the endpoint** via a Dart define or environment variable:
   ```bash
   flutter run \
     --dart-define=MARYTTS_ENDPOINT=https://my-marytts.com/api/tts
   ```
   The `MaryTtsProvider` reads `MARYTTS_ENDPOINT` using `String.fromEnvironment`.
2. No API key is required unless your server enforces authentication â€“ in that case you would need to modify the provider to add headers.
3. The provider is automatically instantiated by `TtsProviderRegistry` and will appear in the dropdown as **MaryTTS Server**.

## Verification
- Select **MaryTTS Server** from the provider list.
- Press **Speak** with some text.
- You should receive audio generated by your MaryTTS server.
